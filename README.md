# Internet Fossils

Welcome to Internet Fossils, a collection of very old websites that are still running to this day, along with a bunch of web assets from very old websites. Explore the internet's history and witness the evolution of web design and content over the years.

## What are Internet Fossils?

Internet Fossils are websites that have been online for a significant period, often since the early days of the internet, and have retained their original design, content, or functionality. These websites provide a glimpse into the past, showcasing the early stages of web development and design.

## Contributing

If you know of any other Internet Fossils that should be included in this collection, feel free to contribute! Simply fork this repository, add the relevant details to the list, and submit a pull request. Make sure that if you contribute, you import the bookmark file directly into a browser of your choice. Then, you can make edits from there and export again as HTML. When updating the README to reflect the changes made in the HTML document. You can use an [HTML to Markdown Converter](https://codebeautify.org/html-to-markdown) to make the process much easier.
